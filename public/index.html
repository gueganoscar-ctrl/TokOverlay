Comment obtenir votre clé API

💡 Bon à savoir : Si TokOverlay est 100% gratuit, c'est grâce à ce service d'API externe qui est entièrement gratuit !
  1. Rendez-vous sur le site eulerstream.com.
  2. Cliquez sur le bouton pour obtenir la clé API.
  3. Créez un compte gratuitement.
  4. Rendez-vous ensuite sur eulerstream.com/dashboard.
  5. Cliquez sur le bouton pour créer votre première clé.
  6. Copiez la clé API générée et collez-la dans votre inscription TokOverlay.
